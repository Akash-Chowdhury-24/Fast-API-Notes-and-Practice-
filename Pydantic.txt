🔷 What is Pydantic?

Pydantic is a Python library used for:

✅ Data validation

✅ Data parsing

✅ Type enforcement

✅ Automatic error handling

It makes sure the data coming into your API is correct and structured.

🔥 In Simple Words

Think of Pydantic as:

"A bodyguard for your API inputs and outputs."

It checks:

Is price really a float?

Is title a string?

Are required fields missing?

Is the format correct?

If something is wrong → it automatically throws a validation error.

🧠 Why FastAPI Uses Pydantic

FastAPI is built on top of Pydantic for:

Request body validation

Response validation

Automatic documentation

Type safety

That’s why FastAPI feels so clean and structured.

📦 Basic Example
from pydantic import BaseModel

class Book(BaseModel):
    title: str
    author: str
    price: float


Now if someone sends:

{
  "title": "Python",
  "author": "Akash",
  "price": "five hundred"
}


Pydantic will automatically return:

{
  "detail": [
    {
      "loc": ["body", "price"],
      "msg": "value is not a valid float",
      "type": "type_error.float"
    }
  ]
}


You didn’t write any validation logic.
Pydantic handled everything 🔥

🔍 What Pydantic Actually Does

When data comes in:

Reads type hints (str, int, float)

Validates them

Converts if possible

Throws structured errors if invalid

Example:

If user sends:

{
  "price": "500"
}


Pydantic automatically converts it to:

price = 500.0


Smart parsing 👌

🧱 Why Not Just Use a Dictionary?

Without Pydantic:

data = request.json()


You would need to manually check:

if "title" not in data:
    return error

if not isinstance(data["price"], float):
    return error


That becomes messy and unsafe.

Pydantic removes all that boilerplate.

🧠 Important Concepts in Pydantic
1️⃣ BaseModel

All models inherit from this.

class User(BaseModel):
    name: str
    age: int

2️⃣ Optional Fields
from typing import Optional

class Book(BaseModel):
    title: str
    description: Optional[str] = None

3️⃣ Field Validation
from pydantic import Field

class Book(BaseModel):
    price: float = Field(gt=0)


Now price must be greater than 0.

4️⃣ Automatic Swagger Docs

Because of Pydantic, FastAPI generates:

Field types

Required fields

Validation rules

Example schemas

Automatically in /docs.
